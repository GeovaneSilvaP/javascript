Calculadora de IMC

Descrição

Calculadora de IMC (Índice de Massa Corporal) com um design simples e moderno, em tons escuros. No topo, há o título “Calculadora de IMC”. Abaixo, aparecem dois campos de entrada: um para altura (com exemplo “1,75”) e outro para peso (com exemplo “70,5”).

Na parte inferior, há dois botões: um vermelho com o texto “CALCULAR” e outro cinza com o texto “LIMPAR”, que provavelmente servem para calcular o IMC com os dados inseridos ou limpar os campos, respectivamente. O fundo escuro realça os elementos da interface, deixando o visual limpo e elegante.
